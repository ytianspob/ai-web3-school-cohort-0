| 角色                       | 职责            |
| ------------------------ | ------------- |
| 用户                | 发起测试需求、支付费用   |
| AI Agent | 自动执行 API 测试任务 |
| 验收者           | 验证测试结果是否达标    |
| 合约                | 暂时托管资金        |
| 仲裁模块   | 处理争议          |
| 链上记录系统                   | 保存执行与支付记录     |

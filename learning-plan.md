# Learning Plan - AI × Web3 School Cohort 0

## 👤 学习目标定位
我是一个技术入门阶段的学习者（偏小白），本次学习目标不是追求复杂开发，而是：

- 理解 AI × Web3 的基本概念体系
- 能够完成基础链上操作（钱包 / 测试网 / 转账）
- 能理解 Agent / Prompt / Workflow 的基本逻辑
- 能完成简单 Vibe Coding 与工具调用实践
- 最终可以做一个最小可运行的 AI × Web3 demo

# 🧭 Week 1｜AI 与 Web3 基础认知 + 实操入门

## 🎯 目标
建立基础概念 + 完成第一次链上与 AI 工具实践

## 📚 学习内容
### AI 部分
- 什么是 LLM（大语言模型）
- Prompt 基本概念（如何和 AI 对话）
- Workflow 是什么（流程化思维）
- Agent 是什么（能执行任务的 AI）
- Tool Use（AI 如何调用工具）

### Web3 部分
- 什么是钱包（Wallet）
- 什么是私钥 / 助记词（安全概念）
- 什么是 Gas fee
- 什么是交易（Transaction）
- 什么是测试网（Testnet）
- 什么是智能合约（Smart Contract）
